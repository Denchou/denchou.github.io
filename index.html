Tic-Tac-Toe

Go to the Repo on Github

Tic-Tac-Toe was my first ever attempt at creating a Single-Page application(SPA). The goal was to create a fully functional Tic-Tac-Toe game as an SPA using authentication. The frontend was built using Javascript, JQuery, Bootleg3, HTML5, and CSS3. Coming from a Computer Science background, creating the logic for Tic-Tac-Toe was the easier part.

It took me awhile to wrap my head around manipulating the DOM using JQuery. Whenever a problem took too long, I resort to googling for solutions -- most of the time it was merely a syntax issue. I had a lot of fun adding in additional features and easter eggs in the end.
